# Задания цикла FOR:
## Задание 1.
Рассматривается множество целых чисел, принадлежащих отрезку [1012; 9638], которые делятся на 7 и оканчиваются на 1.
Найдите сумму таких чисел.
```
s = 0
for i in range(1012, 9639):
 if (i % 7 == 0) and (i % 10 == 1):
    s += i
print(s)
```
Программа должна вывести число 656943.
